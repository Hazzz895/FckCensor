# FckCensor (обход запикивания/цензуры)
Аддон, который автоматически подменивает заблюренные треки, которые есть в [удалённом списке](https://github.com/Hazzz895/FckCensorData/blob/main/README.md), на их незаблюренную версию, а также позволяет подменивать любые треки через контекстное меню
